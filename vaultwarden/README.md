# Vaultwarden
https://github.com/dani-garcia/vaultwarden

### Command to fetch script
```
fetch https://raw.githubusercontent.com/tschettervictor/bsd-apps/main/vaultwarden/vaultwarden-install.sh
```

Don't forget to
```
chmod +x vaultwarden-install.sh
```
