# Jellyfin
https://jellyfin.org

### Command to fetch script
```
fetch https://raw.githubusercontent.com/tschettervictor/bsd-apps/main/jellyfin-server/jellyfin-server-install.sh
```

Don't forget to
```
chmod +x jellyfin-server-install.sh
```
